## Usage

### Installation des dépendances

```bash
$ npm ci
```

### Mode dev en local

```bash
$ npm start
```

### Build final

```bash
$ npm run build
$ npm run start-prod
```

### Génération PDF

```bash
$ npm run build
$ npm run pdf
```
